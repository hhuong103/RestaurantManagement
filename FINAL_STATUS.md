## 🔄 How to Use

### Quick Path (5 minutes)
```bash
1. pip install qrcode[pil]==7.4.2
2. mkdir -p static/qr_codes
3. python app.py
4. Done! ✅
```

### Full Setup (with testing)
```bash
1. pip install -r requirements.txt
2. mkdir -p static/qr_codes
3. python app.py
4. Test order creation
5. Test QR scanning
6. Verify everything works
```

### Complete Deployment
```bash
1. Review documentation
```
